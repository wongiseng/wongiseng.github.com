  • SNE Editor Google App Engine based Editor for editing instance of OWL Ontology, designed for Cinegrid Ontology but extendable for other ontologies.

    This editor can be used to instantiate OWL classes, defines its data properties and configure its object properties. Forms will be generated for filling data properties related to an OWL Classes. Object properties will be represented as input and output port. When the object property has an OWL class as a domain (or range) it will be come an output (or input) port of the OWL Class.

    Existing OWL Ontology needs to be first loaded into Sesame repository which will be used to perform SPARQL queries for generating forms.
